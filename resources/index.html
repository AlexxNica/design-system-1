Sketch

A Sketch template, as well as a Craft Library are available via OneDrive.

View Sketch Resources

How to set it up

In order to use the full feature set of the Craft Library or start from the latest version of the Sketch template, it is important to sync the shared OneDrive folder to your computer. However, syncing with OneDrive isn't quite as simple as Google Drive, so bear with us through the next few steps.

Syncing a Shared OneDrive folder

  1. Make sure that your version of OneDrive was downloaded from the Managed Software Center. Rackspace has built an update to the program that enables us to sync shared folders, so If you download it from another source it will not work. If your computer does not have the Managed Software Center installed on it, you'll need to go to O'Fixies to install it.
  2. Visit the link above and click the "Sync" button.
  3. Syncing the folder will create a second OneDrive folder on your computer. It will also add the folder as a second location to your OneDrive App. TIP: DO NOT rename these folders.

How to use it

Sketch Template

If you choose the Sketch template you can create a new file from it and edit it as you wish without affecting the original document. Sketch maintains the link to the template in your "New from Template" menu. So if you selected it from the shared drive above, it will always access the most up to date file.

Craft Library

If you aren't familiar with Craft, it is a Sketch add-on that includes a feature for shared libraries. If imported into your project the Craft library will automatically alert you when new assets are available.
